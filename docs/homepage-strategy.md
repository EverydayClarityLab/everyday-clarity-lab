# Homepage Strategy

## Purpose
The homepage introduces me first as a product leader and builder, while positioning Everyday Clarity Lab as the space where I explore and create practical AI-supported products, workflows, and ideas.

## Current Homepage Goals
- Establish a clear personal brand
- Present a polished and modern first impression
- Show the intersection of product strategy, guided workflows, and AI-supported building
- Create a strong foundation for future Labs, Kits, Lab Notes, and Contact subpages

## Visual Direction
- Deep navy background
- Teal accents for emphasis and actions
- White typography
- Editorial, minimal, modern presentation
- Clean spacing and restrained visual effects

## Content Structure
- Hero with profile image and atmospheric visual treatment
- Introduction positioning me as a product manager
- Core capabilities section:
  - Product Strategy
  - Guided Workflows
  - AI-Supported Building
- Everyday Clarity Lab section
- How I Work section
- Let’s Talk / LinkedIn section
- Footer

## Release Intent
This first commit should capture only the homepage and the files required for it to render correctly. Subpages and later experiments should be committed separately to preserve a clean product history.

## Commit Boundary
Include only:
- Homepage page implementation
- Shared layout and global styling required by homepage
- Shared components currently used by homepage
- Homepage image assets
- Homepage strategy documentation

Exclude:
- Incomplete subpages
- Experimental code
- Temporary files
- Local build artifacts
